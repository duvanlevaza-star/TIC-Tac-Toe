# Tic Tac Toe - Java Web

Aplicación web desarrollada en Java usando Servlets y JSP.

## Características

- Juego contra la computadora
- Inicio del usuario o del computador
- Detección de ganador
- Detección de empate
- Versión 4 en raya

## Tecnologías

- Java
- Jakarta Servlet
- JSP
- JSTL
- WildFly
